# 一、项目简介
本项目是基于Spring，SpringMVC和Mybatis开发以maven方法构建的图书管理系统，主要功能包括用户操作界面和管理员界面，有图书的增删改查操作，人脸识别登录和验证码登录功能;

# 二、环境配置
idea  
mysql 5.7  
jdk8  
Tomcat 8.5.47
